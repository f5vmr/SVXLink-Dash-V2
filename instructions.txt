#### Instructions for adding the streaming ####
For most people using the software install you shouldl have already a Loopback
soundcard - test it in the terminal $: aplay -l

  is what you should see.

  Now in your svxlink.conf whether you are a SimplexLogic User or RepeaterLogic
  go to TX=Tx1 and change this to TX=MultiTx
  You should have a section 
  
  [MultiTx]
  TYPE = Multi
  TRANSMITTERS = Tx1,TxStream 

  If not - then create it.

  Then locate or create this section.
  [TxStream]
  TYPE = Local
  AUDIO_DEV = alsa:plughw:Loopback,0,0
  AUDIO_CHANNEL = 0
  PTT_TYPE = NONE
  TIMEOUT = 7200
  TX_DELAY = 0
  PREEMPHASIS = 0

  If you have no Loopback card visible then 
  sudo modprobe snd_aloop

  and then sudo nano /etc/loopback.conf

  and copy this to it.

  pcm.loophw {
    type hw
    card Loopback
    device 0
    subdevice 0
}

pcm.loopout {
    type plug
    slave.pcm "loophw"
}
and save it.

then sudo alsactl nrestore.

sudo systemctl restart svxlink svxlink-node and your dashboard should be complete
and display Ver 2.3 Dashboard on the browser tab.
